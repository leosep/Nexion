# Nexion
Una aplicación web de citas construida con **ASP.NET Core** y **Dapper**. Este proyecto permite a los usuarios crear perfiles, buscar personas por intereses, enviar "me gusta" y chatear cuando hay un **match** mutuo.

---

### **Características** 

* **Gestión de Perfiles**: Crea y edita tu perfil de usuario con biografía e intereses.
* **Sistema de "Me Gusta"**: Expresa interés en otros perfiles.
* **Lógica de Match Mutuo**: El chat solo se activa cuando ambos usuarios se dan "me gusta".
* **Chat en Tiempo Real**: Chatea con tus matches usando **SignalR**.
* **Filtrado de Búsqueda**: Encuentra perfiles basándote en intereses específicos.

---

### **Tecnologías Usadas** 

* **Backend**: ASP.NET Core
* **Base de Datos**: PostgreSQL
* **ORM**: Dapper
* **Frontend**: Razor Pages, HTML5, CSS3, JavaScript
* **Estilos**: Tailwind CSS
* **Comunicación en Tiempo Real**: SignalR
